# 소개.

 Spring Framework 기반으로 4년째 웹 어플리케이션을 개발/배포/운영하고 있습니다. 

새로운 것을 마주하는 것에 두려움이 없습니다. 

확장성이 좋고 유지 보수하기 쉬운 코드를 작성하기 위해 항상 고민합니다.

편리함을 제공해주는 ‘어떤 것’을 개발할 때 가장 행복합니다. 클라이언트가 서비스를 실제로 사용하며 들려주는 긍정적인 피드백들은 저의 개발 생활에 있어서 가장 큰 원동력입니다. 

<br>

# 기술.

 소규모 개발 인원으로 신규 솔루션 개발을 진행해왔기 때문에 모든 제작 프로세스에 참여하여 프로젝트의 전반적인 흐름을 파악하고, 한 분야에만 국한되지 않고 다른 분야의 입장을 고려하여 개발할 수 있는 역량을 갖추었습니다. 또한 개발과 운영을 병행하며 다방면으로 견문을 넓혔습니다.

- **Back-end _** `java`, `Spring framework`, `MyBatis`, `Postgresql`
- **Front-end** _ `Javascript`, `jQuery`, `HTML`, `CSS`

 Spring framework 기반으로 웹 애플리케이션을 개발하고 운영하였습니다. Java를 활용하여 필요한 서비스를 명세, 설계하고 구현할 수 있습니다. 데이터베이스는 대용량 데이터에 주로 사용되는 PostgreSQL을 활용할 수 있습니다.

 다양한 RESTful API를 사용하여 데이터를 주기적으로 수집하고 필요한 정보로 가공하여 제공할 수 있습니다.

 솔루션들의 알림 기능을 위한 통합 메시지 서버를 구축하고 REST API를 개발하여 여러 솔루션에 적용하고 운영한 경험이 있습니다.

 고객사의 내부 시스템과 연동하기 위해 인터페이스에 맞춰 통신 모듈과 서비스를 개발한 경험이 있습니다.

 고객의 요구 사항을 반영하여 필요한 기능들을 명세하고 적절한 UI로 스토리보드를 작성하고 Javascript, jQuery,  ajax, HTML, CSS 등을 활용하여 직접 화면을 개발할 수 있습니다.
